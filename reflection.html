Refleksjon

Vår læringsprosess og refleksjoner rundt praksisprosjektet.

Hva vi har lært

Tekniske ferdigheter

Gjennom dette prosjektet har vi fått dybdeforståelse av moderne webutvikling. Vi har lært viktigheten av semantisk HTML, effektiv CSS-organisering og responsive design prinsipper.

Spesielt verdifullt var det å lære om CSS Grid og Flexbox for layout, samt viktigheten av tilgjengelighet i webdesign.

Prosess og samarbeid

Vi har forstått verdien av god planlegging og strukturert arbeidsflyt. Bruken av versjonskontroll (Git) og prosjektdokumentasjon har vært avgjørende for suksess.

Iterativ design og kontinuerlig testing har hjulpet oss å bygge en bedre brukeropplevelse.

Utfordringer og løsninger

Responsive design kompleksitet

Utfordring: Å få layouten til å fungere perfekt på alle skjermstørrelser.

Løsning: Brukte mobile-first tilnærming og grundig testing på ulike enheter.

Performance optimalisering

Utfordring: Nettsiden lastet for sakte på mobile enheter.

Løsning: Optimaliserte bilder, minifiserte CSS og brukte effektive selektorer.

Tilgjengelighet

Utfordring: Å sikre at nettsiden er tilgjengelig for alle brukere.

Løsning: Implementerte ARIA-etiketter, fokus-indikatorer og semantisk HTML.

Fremtidige forbedringer

  • Implementere JavaScript for interaktive elementer
  • Legge til animasjoner og mikrointeraksjoner
  • Integrere med backend for dynamisk innhold
  • Implementere mørk modus
  • Forbedre SEO og metadata
  • Legge til kontaktskjema med validering
